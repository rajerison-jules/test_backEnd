# Pour tester
cette API est faite avec Node v14.16.1  avec le SGBDDR Mysql
pour le lancer vous devez installer Node et puis Mysql
# cree une utilisateur root  
```bash
mysql -u root -p
```

cree un base de donne sous le nom de 

* testdb
```bash
CREATE DATABASE testdb
```
### lancer le serveur

```bash
node server.js
```
